Exercise 1¶

Let $\theta_1$ and $\theta_2$ be real valued parameters of the model

$$ y = \frac{\theta_1 x}{\theta_2 + x}. $$

a. Choose two suitable prior distributions for $\theta_1$ and $\theta_2$ and use HMC algorithm to find their posterior distributions, conditioning on the observations

$$ x=(28,55,110,138,225,375)\\ y=(0.053,0.060,0.112,0.105,0.099,0.122). $$

b. Discuss how different parameters for both priors and the HMC algorithm lead to different estimates.

c. Plot the most reliable posterior distributions, according to convergence checks on the traces.

Solution:

a, b. Given

$$ y = \frac{\theta_1 x}{\theta_2 + x} $$

I consider the following model:

  • Given as known $\theta_1$ $\theta_2$ and x, the probability to having a y is distributed as a normal centered on $ y(x) = \frac{\theta_1 x}{\theta_2 + x} $ and $\sigma = 1$:

    $$y | \theta_1, \theta_2, x \sim \mathcal{N}(y(x),1) $$

  • We assume for the prior distribution $\pi(\theta_1, \theta_2 | x)$ that, given the observation x, the random variables $\theta_1$ and $\theta_2$ are indipendent:

    $$\pi(\theta_1, \theta_2 | x) = \pi(\theta_1 | x) \pi(\theta_2 | x) $$

  • Then, by applying Bayes' Theorem we end up with

    $$ p (\theta_1, \theta_2 | y, x ) = \frac{p(y | \theta_1, \theta_2) \pi(\theta_1 | x) \pi(\theta_2 | x) }{p(y,x)} $$

I decide to use the following three distributions for the priors of $\theta_1$ and $\theta_2$:

  • Gamma Distribution
  • Exponential Distribution
  • LogNormal Distribution

I justify this assumption by the fact that I consider that both $\theta_1$ and $\theta_2$ positives. All choices of the hyperparameters are taken in order to have a distribution that could take very likely a broad range of values, since I have no specifications about $\theta_1$ and $\theta_2$.

  • $\theta_1 \sim Gamma(7.5, 1.0) $ and $ \theta_2 \sim Gamma(7.5, 1) $.

def scale(x = x_data):
    
    # models weight in terms of the initial guess
    theta_1 = pyro.sample("theta_1", dist.Gamma(7.5,1))
    theta_2 = pyro.sample("theta_2", dist.Gamma(7.5,1))
    yhat = (theta_1*x)/(theta_2+x)
    #models measurment process
    y = pyro.sample("y", dist.Normal(yhat,1))
    
    return y

Exercise 2¶
A bivariate Gibbs sampler for a vector $x=(x_1,x_2)$ draws iteratively from the posterior conditional distributions in the following way:


    

  • choose a starting value $p(x_1|x_2^{(0)})$
    • 

  • for each iteration $i$:
      

    • draw $x_2(i)$ from $p(x_2|x_1^{(i-1)})$
      • 

    • draw $x_1(i)$ from $p(x_1|x_2^{(i)})$
      • 

    

    • 
















a. Supposing that samples are drawn from a bivariate normal distribution

$$
{x_1 \choose x_2} \sim \mathcal{N} \Bigg[ {0 \choose 0} , \begin{pmatrix} 1 & \rho \\ \rho & 1 \end{pmatrix} \Bigg],
$$

implement a Gibbs sampler for $x$ which takes as inputs the number of iterations `iters` and the number of warmup draws `warmup`.




b. Use your implementation of Gibbs sampler to infer the parameters $\theta=(\theta_1,\theta_2)$ from Exercise 1.















Solution:


a. Using the properties of the multivariate normal distribution

$$
x_1|x_2,\: x \text{ ~ } N(\rho x_2,\: 1-\rho^2) \text{ ~ } \rho x_2 + \sqrt{1-\rho^2}N(0,\:1)
$$
and

$$ 
x_2| x_1,\: x \text{ ~ } N(\rho x_1,\: 1-\rho^2) \text{ ~ } \rho x_1 + \sqrt{1-\rho^2}N(0,\:1)
$$
Then, given $\rho$ we can implement the gibbs_sampler for the random vector $x$ distributed as a bivariate normal distribution:

def gibbs_sampler(rho, iters, warmup):
    
    #Initialize the variables 
    x_1 = torch.zeros(warmup + iters, 1)        
    x_2 = torch.zeros(warmup + iters, 1)
    
    x_2[0] = pyro.sample("x2_0", dist.Normal(0,2))
    x_1[0] = pyro.sample("x1_0", dist.Normal(rho*x_2[0].item(), np.sqrt(1-rho**2)))
                         
    for t in range(1,warmup+iters-1):
        x_2[t] = pyro.sample("x2_"+str(t), dist.Normal(rho*x_1[t-1].item(), np.sqrt(1-rho**2)))
        x_1[t] = pyro.sample("x1_"+str(t), dist.Normal(rho*x_2[t].item(), np.sqrt(1-rho**2)))
        
    x1 = x_1[warmup:iters]
    x2 = x_2[warmup:iters]
    
    return x1, x2

def gibbs_sampler_mu(mu, rho, iters, warmup):
    
    #Initialize the variables 
    x_1 = torch.zeros(warmup + iters, 1)        
    x_2 = torch.zeros(warmup + iters, 1)
    
    x_2[0] = pyro.sample("x2_0", dist.Normal(0,5))
    x_1[0] = pyro.sample("x1_0", dist.Normal(mu[1].item()+rho*x_2[0].item(), np.sqrt(1-rho**2)))
                         
    for t in range(1,warmup+iters-1):
        x_2[t] = pyro.sample("x2_"+str(t), dist.Normal(mu[0].item()+rho*x_1[t-1].item(), np.sqrt(1-rho**2)))
        x_1[t] = pyro.sample("x1_"+str(t), dist.Normal(mu[1].item()+rho*x_2[t].item(), np.sqrt(1-rho**2)))
        
    x1 = x_1[warmup:iters]
    x2 = x_2[warmup:iters]
    
    return x1, x2
